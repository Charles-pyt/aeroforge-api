AeroForge API

The aerospace validation and telemetry engine.

GET /parts

Returns the catalog of standard aerospace components.

GET /telemetry

Returns simulated live telemetry for tracking interfaces.

POST /validate

Validates a CAD design against strict engineering tolerances.

Example Request:

curl -X POST http://localhost:8080/validate \
  -H "Content-Type: application/json" \
  -d '{"name":"Ariane6_Booster","geometry":"demi-cone avec un rectangle au bout"}'
